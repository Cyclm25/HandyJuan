# HandyJuan

HandyJuan is a FlutterFlow and Firebase-powered hyperlocal service platform that helps local workers gain visibility, build trust, and connect with nearby residents through real-time bookings, ratings, certifications, and responsiveness.

## Problem

Many local service workers such as tuberos, electricians, cleaners, drivers, tutors, and manikurista are skilled and actively looking for work, but they are not well represented in formal job platforms.

Most rely on word of mouth, Facebook posts, or random referrals, which often lack structure, visibility, and trust.

At the same time, residents who need help often struggle to find nearby and reliable workers.

## Solution

HandyJuan provides a simple platform where residents can find nearby workers, while local workers can showcase their services and receive booking requests.

The platform works both ways:
- Residents can find reliable nearby services
- Workers can become visible, build reputation, and earn more consistently
- Users being a user can also offer a service

## Key Features

- Hyperlocal worker discovery based on nearby location
- Worker profiles with skills, service details, and certifications
- Ratings and reviews for trust and credibility
- Responsiveness indicators for better decision-making
- Real-time booking requests
- Dual-role system where users can both hire and offer services

## Tech Stack

- FlutterFlow – app design and development
- Firebase Authentication – user login and account management
- Cloud Firestore – real-time database for users, bookings, profiles, and ratings
- Firebase Storage – storing profile images, certifications, and service photos

## Impact

HandyJuan transforms existing local skills into visible, trusted, and accessible livelihood opportunities. It helps workers receive more consistent service requests while helping residents find reliable help within their community.
